07
Usage Rules
Do and Don't
These rules protect the mark. Every violation weakens what we are building. When in doubt, use the approved files — do not recreate the mark.
Do
Use the mark on its approved background only — dark on dark, white on white.
Always maintain clear space equal to the height of the "M" in MOXSUITE on all four sides.
Use the gradient wordmark on dark backgrounds. Use the dark wordmark on light.
Use the family dot on all product marks — it is part of the system, not optional.
Switch to Arabic RTL layout when displaying in Arabic. Use Noto Sans Arabic.
Use only the four approved versions: Dark, Black, Light, and White.
Scale the mark proportionally. The icon and wordmark always appear at the same ratio.
Don't
Never recreate the Nexus mark. Use only the approved SVG files.
Never alter the brand colours. Do not substitute Pantone, CMYK, or approximate values.
Never stretch, skew, rotate, or distort the mark in any direction.
Never add a drop shadow, glow, emboss, or any effect to the mark.
Never place the mark on a patterned, photographic, or coloured background without approval.
Never display the wordmark without the icon mark on primary brand applications.
Never use the tagline without proper capitalisation: The Enterprise Operating System — not THE ENTERPRISE OPERATING SYSTEM.

Clear Space & Minimum Size
Protection Zone
The Nexus mark must breathe. No other element — text, imagery, another logo, a border — may enter the clear space zone. The minimum size ensures the mark is always legible and never compromised.
Clear space = 1× M height
Digital minimum
32px
Icon mark only. Below this, use text only.
With wordmark minimum
120px
Full lockup. Below 120px, use icon only.
Print minimum
15mm
Icon mark in print. Full lockup at 40mm+.
